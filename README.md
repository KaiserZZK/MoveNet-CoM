#  MoveNet-CoM

Center of mass identification model built with MoveNet for 
athletic video analysis 

## Creating the virtual environment and installing dependencies
```shell
conda env create -f environment.yml
```
To activate the environment, execute the following command:
```shell
conda activate venv_MoveNet_test1 
```
to make it show up on our kernel 
```shell
ipython kernel install --user --name=venv_MoveNet_test1
```
find `venv_MoveNet_test1` in your Jupyter Notebook interface  

(include a screenshot)

restart the kernel if needed 

### You may also find those commands useful: 

To update packages
```shell
conda env update --file environment.yml --prune
```
To check packages 
```shell
conda list 
```
